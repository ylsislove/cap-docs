# 基本参数介绍

# 楼层高度

设置建筑物的单层楼层高度，默认为3米

# GeoHash大小

GeoHash是一种空间点索引算法，值越大，表示索引精度越高。默认为8

# StepRadius

半球分析的间隔步长，默认是50米

# MaxRadius

半球分析的最大分析半径，默认是1000米

# Is Normal

分析结果是否归一化，默认为否
